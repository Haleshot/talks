# Stop reprocessing everything

Rust Delhi Meetup talk about incremental data pipelines with [CocoIndex](https://github.com/cocoindex-io/cocoindex).

**Date:** January 17, 2026

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3030

## Export to PDF

```bash
npm run export
```
